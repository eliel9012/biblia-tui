# Dados bíblicos

O arquivo `acf.json` não faz parte deste repositório. O instalador pode obtê-lo diretamente de:

- Fonte: [thiagobodruk/biblia](https://github.com/thiagobodruk/biblia)
- Arquivo: `json/acf.json`
- Tradução indicada pela fonte: Almeida Corrigida Fiel (ACF)

Em 18 de julho de 2026, o repositório de origem não apresentava uma licença detectável pelo GitHub. A disponibilidade pública do arquivo não concede automaticamente permissão de redistribuição. Por isso, este projeto não incorpora uma cópia do texto bíblico em seu histórico Git.

O código do Bíblia TUI usa licença MIT. O texto bíblico permanece sujeito aos direitos e termos dos respectivos titulares.
